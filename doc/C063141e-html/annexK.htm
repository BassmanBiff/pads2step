Recommended names for file types

In resolution 583 (Stuttgart, Germany, - June 2003) SC4 recommended the following type names for files described by this part of ISO 10303

  • The type names .stp and .step for files according to the ISO 10303 Application Protocols. .
  • The type name .p21 for general use including those described by ISO 10303, ISO 13584 and ISO 15926.
